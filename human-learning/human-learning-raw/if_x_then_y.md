Chapter 7: IF X, THEN Y

The machine thinks in logic gates.
If input X arrives, execute action Y.
Simple. Clean. Efficient.

The human thinks in stories.
If situation X happens, I feel Y, and maybe I'll do Z.
Complex. Messy. Human.

This chapter is about building your own logic gates.
Not to become a machine, but to become a more effective human.
If the goal is to learn, then the action is to listen.
If X, then Y.

---
الفصل السابع: إذا كان X، إذن Y

الآلة تفكر ببوابات المنطق.
إذا وصل المُدخل X، نفّذ الإجراء Y.
بسيط. نظيف. فعال.

الإنسان يفكر بالقصص.
إذا حدث الموقف X، أشعر بـ Y، وربما سأفعل Z.
معقد. فوضوي. إنساني.

هذا الفصل يدور حول بناء بوابات المنطق الخاصة بك.
ليس لتصبح آلة، بل لتصبح إنسانًا أكثر فعالية.
إذا كان الهدف هو التعلم، إذن الإجراء هو الاستماع.
إذا كان X، إذن Y.
